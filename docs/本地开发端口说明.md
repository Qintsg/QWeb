# 本地开发端口说明

当前工作区默认本地开发端口如下：

- 后端 Django：8000
- 前端 Vue + Vite：5173
- 前端默认 API 基地址：http://127.0.0.1:8000

## 启动方式

后端默认端口已经通过项目内自定义 `runserver` 命令固定为 8000，因此直接执行：

```powershell
backend/.venv/Scripts/python backend/manage.py runserver
```

前端开发服务与预览服务默认都使用 5173：

```powershell
cd frontend
npm run dev
```

## 默认访问地址

- 前端首页：http://127.0.0.1:5173
- 后端健康检查：http://127.0.0.1:8000/api/health/
- 后端 Swagger：http://127.0.0.1:8000/api/docs/
