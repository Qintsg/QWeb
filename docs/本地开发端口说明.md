# 本地开发端口说明

当前工作区默认本地开发端口如下：

- 后端 Django：8001
- 前端 Vue + Vite：3001
- 前端默认 API 基地址：http://127.0.0.1:8001

## 启动方式

后端默认端口已经通过项目内自定义 `runserver` 命令固定为 8001，因此直接执行：

```powershell
backend/.venv/Scripts/python backend/manage.py runserver
```

前端开发服务与预览服务默认都使用 3001：

```powershell
cd frontend
npm run dev
```

## 默认访问地址

- 前端首页：http://127.0.0.1:3001
- 后端健康检查：http://127.0.0.1:8001/api/health/
- 后端 Swagger：http://127.0.0.1:8001/api/docs/
